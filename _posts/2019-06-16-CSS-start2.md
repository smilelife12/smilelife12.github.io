## CSS 이어서!
- - -

### CSS의 선택자 알아내기
- - -

\<style> 태그내에 클래스를 새롭게 만들어 줄수 있고, 그 클래스를 통해서 CSS 설정을 해 줄 수 있다. .을 통해서 이름을 설정해 줄 수 있고,
```html
<a href="1.html" class="saw">HTML</a>
```
와 같이 사용할 수있다. 그리고 클래스는 중복으로 설정할 수 있지만, 이보다는 id선택자를 사용하는 것이 더 유용하다. 그리고 여기서 우선순위는 id > class > 태그 라는 순서로 사용된다. 또한, id의 값은 단 한번 등장하고 사용되어야 한다.
이를 style태그에서 정의하는 방법은 .class라거나 #id 와 같은 방식으로 사용한다.
<details markdown="1">
<summary> 예시코드</summary>
``` html
<head>
  <title>WEB - CSS</title>
  <meta charset="utf-8">
  <style>
    #active {
      color:red;
    }
    .saw {
      color:gray;
    }
    a {
      color:black;
      text-decoration: none;
    }
    h1 {
      font-size:45px;
      text-align: center;
    }
  </style>
</head>
<body>
  <h1><a href="index.html">WEB</a></h1>
  <ol>
    <li><a href="1.html" class="saw">HTML</a></li>
    <li><a href="2.html" class="saw" id="active">CSS</a></li>
    <li><a href="3.html">JavaScript</a></li>
```
</details>

### 박스 모델
- - -
style내에서 사용하는 값들의 예시들로
* border  라는 값을 이용해서 박스형태를 설정할 수 있다. 여기서 박스를 설정하고, 추가로 클래스를 ,로 구분해서 여러개를 한번에 쓸 수 있다.
* padding 을 통해서 박스 자체의 크기를 지정해 줄 수 있다.
* margin  은 각 태그 간의 간격을 지정할 수 있다.

크롬에서 f12를 눌러서 개발자 도구를 보면 쉽게 이해할 수 있다.

그리고 이를 이용해 개발을 할 때는 모든 걸 다 암기해서 하려고 하지말고, 크롬의 개발자도구를 잘 활용해서 margin, padding등의 크기를 고려하고, 값을 변경해 스타일을 꾸며준다.

### 그리드
- - -
목록과 본문이 표처럼 나타나게 하는 방법으로 사용할 수 있는 방식이다.
* div 태그는 아무런 의미가 없는 태그로 그냥 사용한다. 유사한 값은 span이 있다. 차이점은 block이냐 inline이냐의 차이이다.

그리드를 사용하기 위해서는 이를 묶는 부모 태그가 필요하며 이에 대한 아이디는 알아서 지정하자. 그를 이용해서 순수하게 스타일을 설정할 수 있는 영역을 만들어준다.

이를 이용해 내용을 무엇을 넣더라도 자동으로 크기가 변동되는 유용한 방법이다. 그리고 이를 사용해도 되나 안되나를 알 수 있는 사이트는 [Can i use](https://caniuse.com) 사이트에서 어디까지 사용되는지 알 수 있다.

여기서 grid를 사용할 때 사용한 id를 이용해 #id \<태그>와 같은 방식으로 현재 사용하는 그리드 내의 내용에 대한 스타일의 편집을 해낼 수 있다.

### 반응형 웹
- - -
웹의 크기에 따라서 모양이 변하도록 설정할 수 있다.
이를 사용하는 방식은 \@media를 이용해서 지정할 수 있고, 예를들어 스크린의 크기가 몇인지에 따라 다르게 스타일을 지정할 수 있다.
그래서 이를 이용해 범위를 설정하고 이에 따라 지정된 태그에서 나타났던 특정 설정을 none으로 설정해서 나타나지 않게하는 방식을 이용할 수 있다.

### CSS 코드 재사용
- - -
style이라는 태그의 코드를 html 마다 옮겨서 작성하는 것은 너무나도 귀찮고, 하기 싫어진다. 이처럼 중복이 많이 발생하는 것을 없애기 위해서 하는 방법은 바로 CSS라는 확장자를 갖는 파일을 만든다. 그리고 head부분에
```html
<link rel="stylesheet" href="style.css">
```
라고 작성해서 사용하주면 된다. 그러면 스타일 태그를 css파일에서 편집하고, 이를 html에서 사용할 수있게 된다. 이를 통해 가독성도 올리고, 편집에 효율성도 올라가게된다.

## 정리하며
- - -
CSS에서 가장 중요한 것은 선택자, 속성이 두개가 가장 중요하다.
이 두가지를 잘 이용할 수 있다면, 웹페이지의 편집은 굉장히 쉬워질 것이다.
