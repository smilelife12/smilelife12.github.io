**알고리즘 문제풀이**

-알고리즘에서 사용한 내용들
  - 파이썬에서 ``__str__`` 함수와 ``__repr__`` 함수의 구분
    - repr은 표현하다는 뜻 그 값을 적절히 문자열로 표현
    - class에서 문자열로 값을 표현할 수 있음
    - ``__str__``이 객체를 평문화하는 데 집중, ``__repr__``은 객체를 표현하는데 집중되어 있다는 차이점이 있음
  - 파이썬에서는 소수점 계산에 오류가 발생하는 경우가 종종있음
    - 이를 해결하기 위해서 `round()`함수를 이용해서 반올림을 통해 해결


**운영체제**

공룡책
Chap1. Introduction
1. OS가 하는 일 -> 각각을 조율하는 역할
  - 유저 입장에서는  쉽게 IO장비를 쉽게 사용하고 리소스 활용
  - 시스템 입장에선 리소스 할당자 취급, 간혹 컨트롤 매니저
  - OS는 컴퓨터가 많은 곳에 포함되고 사용, 하드웨어를 사용하는 app와 IO 장비간의 조율을 위해 만든 SW라고 볼 수 있음
2. 컴퓨터 시스템
  - 최근 시스템은 CPU 와 각종 컨트롤러 어댑터 들이 하나의 메모리를 공유
  - 실행 과정은 ROM 등에 설치된 시작프로그램 혹은 bootstrap 프로그램으로 실행 -> CPU register에서 디바이스 컨트롤러 모든 시스템 등을 초기화 -> bootstrap에 OS 로드 방법, 시스템 실행법을 기억해논 상태로, 메모리 로드 -> 커널이 켜지면 유저들은 시스템 사용
  - 보통 이벤트는 interrupt를 통해 발생, HW는 CPU에 signal, SW는 system call 실행
  - 컴퓨터에서 interrupt가 중요하며, 메커니즘 있고, 이를 처리하는 기법도 중요
  - 보통 테이블에 저장해 처리, 각 interrupt 벡터에 주소처리해 저장
  - 주소 저장시 중단된 명령 주소 저장, 명령 처리 후 다시 가져옴
